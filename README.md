# 4. Project expected effect
도시 및 환경관리자로 하여금 쾌적한 환경계획을 위하여 효율적인 쓰레기 수거계획 및 쓰레기통 배치 계획의 인사이트를 제공하고자 한다. 
사용자로 하여금 쓰레기통이 넘치는 것을 방지하고 직접 쓰레기통에 손을 대지 않아도 사용이 가능하도록 하여 청결하고 안전한 도시에 기여하고자 한다.
